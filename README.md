## 📌 Project Overview

This is the foundation project for the AI Engineering track at DecodeLabs.  
I have built a **rule-based chatbot** that operates using **deterministic logic** — no machine learning, just pure control flow and decision-making.

> *"Before you build systems that learn on their own, you must master the art of teaching a machine through explicit if-else instructions."*

---

## 🎯 Goal

Create a simple rule-based chatbot that responds to predefined user inputs using if-else logic, runs in a continuous loop, and handles greetings and exit commands.
